Yamaha PSR-E473 WAV Exporter

Drop in any audio or video file. Get back a WAV the keyboard will actually load.

Output spec

  • WAV, 44.1 kHz, 16-bit, stereo PCM
  • Filename: ASCII, uppercase, no spaces (e.g. SONG.WAV)
  • Drop on FAT32 USB root → SONG button → dial past internal songs

Click to pick a file, or drop one here.

mp3, m4a, mp4, wav, flac, ogg, webm, aac — anything ffmpeg reads.

Why this exists

The PSR-E473 will only play WAV files that are exactly 44.1 kHz, 16-bit, stereo PCM. Any deviation — 48 kHz, 24-bit, mono, compressed WAV — produces a load error on the keyboard. This page re-encodes any input to the exact spec entirely in your browser using ffmpeg.wasm. Nothing is uploaded.
